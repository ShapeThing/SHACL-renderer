# SHACL-renderer

A React Component which renders a SHACL shape as:

- a form
- a view
- facets
- a JavaScript object
- a TypeScript type

This is a work in progress...

- Supports a subset of SHACL
- Also works without a SHACL shape
- You can add your own widgets
